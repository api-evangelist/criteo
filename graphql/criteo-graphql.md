# Criteo GraphQL API

Criteo is a global commerce media company offering retargeting, audience targeting, and retail media solutions. Their API covers catalog management, campaign creation, ad set targeting, audience segments, and performance reporting.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Criteo REST API
**Documentation:** https://developers.criteo.com/retail-media/docs/welcome-to-criteo
